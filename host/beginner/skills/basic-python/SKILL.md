# Basic Python Skill

## When to Use
Use this skill when you need to:
- Write simple Python scripts
- Understand basic Python syntax
- Avoid common beginner mistakes
- Test and run Python code

## Simple Python Structure

```python
# This is a comment

# Variables
name = "Python"
number = 42

# Print
print(f"Hello {name}")
print(number)

# Simple function
def add(a, b):
    return a + b

result = add(5, 3)
print(result)
```

## Testing Your Code

Always run your code after creating it:
```python
execute(command="python my_script.py")
```

## Common Beginner Mistakes

1. **Forgetting quotes**: `print(hello)` ❌ → `print("hello")` ✅
2. **Wrong indentation**: Python uses spaces/tabs consistently
3. **Misspelled commands**: `exicute` ❌ → `execute` ✅

## Quick Start Example

```python
# Create a simple script
write(file_path="hello.py", content="""
print("Hello, World!")
print("This is my first script")
""")

# Run it
execute(command="python hello.py")
```
