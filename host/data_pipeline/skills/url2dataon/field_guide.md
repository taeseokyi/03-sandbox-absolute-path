# DataON 필드 작성 빠른 참조

> url2dataon 스킬 전용 참조 가이드. 에이전트가 DataON JSON을 작성하기 전 확인한다.

---

## 필수 필드 체크리스트

JSON 작성 후 아래 항목이 모두 채워졌는지 확인한다.

| 필드 경로 | 타입 | 조건 |
|-----------|------|------|
| `기본.국내외` | `"DOMESTIC"` \| `"ABROAD"` | 국내 기관이면 `"DOMESTIC"` |
| `기본.제목_주언어` | `str` | 한국어 제목 (비어 있으면 영어 제목 번역 필요) |
| `기본.설명_주언어` | `str` | 한국어 설명 (비어 있으면 영어 설명 번역 필요) |
| `기본.키워드_주언어` | `list[str]` | 한국어 키워드 1개 이상 |
| `기본.과학기술표준분류` | `list[str]` | NTIS 분류 1개 이상 |
| `인물정보` | `list` | `"역할": "CREATOR"` 인물 1명 이상 |
| `파일.출처URL` | `list[str]` | 소스 URL 포함 (또는 `파일목록` 중 하나) |

---

## 언어 규칙

| 원문 언어 | 주언어 필드 처리 | 부언어 필드 처리 |
|-----------|----------------|----------------|
| 한국어 원문 | 원문 그대로 `제목_주언어`, `설명_주언어`, `키워드_주언어` | 영어 있으면 `제목_부언어`, `설명_부언어`, `키워드_부언어` |
| 영어 원문 | 번역하여 `제목_주언어`, `설명_주언어`, `키워드_주언어` 채움 | 원문 영어 → `제목_부언어`, `설명_부언어`, `키워드_부언어` |

---

## 날짜 형식

모든 날짜는 `"yyyy-mm-dd"` 문자열. 연도만 있으면 `"yyyy-01-01"`.

---

## 인물 역할 값 (`인물정보[].역할`)

| 값 | 의미 |
|----|------|
| `"CREATOR"` | 생성자 — 데이터 작성자 (필수 1명) |
| `"MANAGER"` | 담당자 |
| `"CONTRIBUTOR"` | 기여자 |

---

## 라이선스 매핑 (`공개설정.라이선스종류`)

| 페이지 표시 | JSON 입력 값 |
|------------|-------------|
| CC BY | `"저작자표시"` |
| CC BY-NC | `"저작자표시-비영리"` |
| CC BY-ND | `"저작자표시-변경금지"` |
| CC BY-SA | `"저작자표시-동일조건변경허락"` |
| CC BY-NC-SA | `"저작자표시-비영리-동일조건변경허락"` |
| CC BY-NC-ND | `"저작자표시-비영리-변경금지"` |
| 표시 없음 | `"저작자표시"` (기본값으로 사용) |

---

## 과학기술표준분류 선택 기준

기관 분야·데이터 키워드를 참고하여 NTIS 분류 중 가장 적합한 항목 선택.

| 기관·데이터 분야 | 대표 분류 예시 |
|----------------|--------------|
| 극지·해양·수산 | `"해양학"`, `"환경과학"`, `"기후학"` |
| 지질·자원·광물 | `"지질학"`, `"지구물리학"`, `"자원공학"` |
| 에너지·원자력 | `"에너지"`, `"신재생에너지"`, `"원자력공학"` |
| 항공·우주·천문 | `"항공우주공학"`, `"천문학"` |
| 생명·의학·보건 | `"생물학"`, `"의학"`, `"보건학"` |
| 정보·통신·SW | `"정보통신"`, `"컴퓨터과학"` |
| 건설·교통·환경 | `"토목공학"`, `"환경공학"` |
| 기타 | `"기타"` |

## 수집지역 유형 (`추가.데이터수집지역[]`)

좌표 정보가 있을 때만 기입한다.

```json
// Point — 위·경도 1쌍
{
  "유형": "geoLocationPoint",
  "수집지역정보": { "위도": 37.5, "경도": 129.0 }
}

// Place — 지역명만
{
  "유형": "geoLocationPlace",
  "수집지역정보": { "지역명": "남극 장보고 기지" }
}

// Box — 4좌표 (좌하단→우하단→우상단→좌상단 순)
{
  "유형": "geoLocationBox",
  "수집지역정보": {
    "좌표목록": [
      {"위도": 60.0, "경도": 120.0},
      {"위도": 60.0, "경도": 130.0},
      {"위도": 70.0, "경도": 130.0},
      {"위도": 70.0, "경도": 120.0}
    ]
  }
}
```

---

## 수집기간 (`추가.데이터수집기간[]`)

데이터 수집 시작·종료 날짜가 있을 때 기입한다.

```json
{
  "시작일자": "2020-01-01",
  "종료일자": "2024-12-31"
}
```

---

## NTIS 과제 연관 정보 (`연관.과제목록[]`)

NTIS 과제번호(**숫자 10자리**, 예: `1525008151`)가 있을 때만 기입한다.

```json
{
  "관계유형": "isOutputOf",
  "식별자유형": "NTIS",
  "식별자": "1525008151",
  "과제명_한글": "...",
  "부처명": "...",
  "과제수행기관": "...",
  "과제책임자": {
    "책임자명_한글": "...",
    "책임자명_영문": null,
    "기관명_한글": null
  },
  "상세입력": {
    "참여자목록": [],
    "연구기간_시작일": "yyyy-mm-dd",
    "연구기간_종료일": "yyyy-mm-dd",
    "총연구비": null,
    "키워드_한글": [],
    "키워드_영문": []
  }
}
```

## 공개 설정 기본값

| 필드 | 기본값 | 비고 |
|------|--------|------|
| `공개설정.공개구분` | `"PUBLIC"` | 즉시공개 |
| `공개설정.DOI출판` | `true` | 승인 후 DOI 자동 발급 |
| `공개설정.라이선스종류` | `"저작자표시"` | CC 정보 없을 때 기본값 |
